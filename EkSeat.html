EkSeat.com- A Ride Sharing WebApp

Key Points

  • Real-time face recognition system
  • Secure access control with ESP32Cam
  • Machine learning algorithms for accurate identification
  • Solenoidal door lock integration
  • Works in various lighting conditions
  • Easy to integrate with existing systems

Description

Developed a face recognition based door lock system using the ESP32Cam and Solenoidal Door Lock System. The model utilizes machine learning algorithms to accurately identify authorized individuals and grant access.

The system was designed to work reliably in different environmental conditions, with special attention given to varying lighting scenarios that typically challenge facial recognition systems.

This project demonstrates the practical application of AI and IoT technologies in creating secure, automated access control systems that can be deployed in residential, commercial, or institutional settings.
