Face Recognition Model

Overview

Developed a face recognition based door lock system using the ESP32Cam and Solenoidal Door Lock System. The model utilizes machine learning algorithms to accurately identify authorized individuals and grant access.

Technologies Used

  • ESP32Cam
  • Solenoidal Door Lock System
  • Machine Learning Algorithms
  • Python

Features

  • Real-time face recognition
  • Secure access control
  • Easy integration with existing door lock systems

Challenges Faced

Implementing accurate face recognition in varying lighting conditions and ensuring the security of the system against unauthorized access.

Future Improvements

Enhancing the model's accuracy and speed, as well as adding features such as remote access control and logging of access attempts.
